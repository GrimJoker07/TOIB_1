# TOIB_1
Вариант 2 Напишите программу на Python, которая определяет функцию для генерации случайных паролей указанной длины. Функция принимает необязательный параметр length, который по умолчанию равен 8. Если длина не указана пользователем, пароль будет состоять из 8 символов.

Для запуска можете нажать Ctrl + A, после прожать ПКМ и выбрать функцию "Run Code". Также можно попробовать нажать кнопку запуска в правом верхнем углу.
